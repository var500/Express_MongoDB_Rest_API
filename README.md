## **Installation**

**Cloning the repo**

```bash
git clone https://github.com/var500/Express_MongoDB_Rest_API.git
```

**Installing the dependencies**

```bash
npm install
```

**Starting the localhost-server**

```bash
npm run dev
```

**Creating the build**

```bash
npm run build
```
